Developer tools you might need to use:

caniuse.com

This is a website, that allows you to search for whatever CSS method or attribute you want and make sure that the browsers support it.

w3schools.com

W3Schools is a web developers site, with tutorials and references on web development languages such as HTML, CSS, JavaScript, PHP, SQL, Python, jQuery, Java, W3.CSS, and Bootstrap, covering most aspects of web programming.

browserling.com

Live interactive cross-browser testing. To check if your work is actually compatible the way you want it to be, use a tool like browserling to simulate your work in older environments.
